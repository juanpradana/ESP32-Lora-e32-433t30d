# ESP32-Lora-e32-433t30d
This repository for ESP32 to communication eith other esp32 using Lora e32-433t30d.

![Lora e32-433t30d](https://www.ebyte.com/Uploadfiles/Picture/2021-5-25/20215251730359457.jpg)

# Wiring
ESP32 >>  Lora e32-433t30d

GND   >> GND

GND   >> M0

GND   >> M1

D16   >> Tx

D17   >> Rx

3.3V  >> VCC

# Explanation
Rx Tx must swap.
